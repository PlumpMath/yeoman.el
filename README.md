## Installation

    $ cd ~/.emacs.d/vendor
    $ git clone git://github.com/julienXX/yeoman.el.git

In your Emacs config:

    (add-to-list 'load-path "~/.emacs.d/vendor/yeoman.el")
    (require 'yeoman)
